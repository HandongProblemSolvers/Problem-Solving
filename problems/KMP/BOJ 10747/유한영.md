**개념 정리**

1. KMP 알고리즘 
KMP 알고리즘은 순차적으로 검색해나가지만, 이미 검색된 부분을 더 이상 다시 확인 하지 않을 수 있도록 처리를 해줌

prefix와 postfix를 기준으로 pi를 만드는데, Search 하려는 문자열의 각 원소를 기준으로 prefix와 postfix가 몇개나 같은 지를 확인 후 별도의 int배열 pi에 저장
그리고 Search 당하는 문자열은 pi를 기준으로 검색함. 
중복으로 검색하는 부분을 최대한 줄여줄 수 있도록 pi를 기준으로 문자열 검색중 틀릴 시 해당 부분을 건너뛰어서 검색함
