- **내가 이해한 문제내용**
  - 앞으로 지그재그로 젤 길게 가즈아!!

- **접근 방식**
  - 페어 배열을 정렬하고,
  - for(int i=0;i<n;i++){
  -      ll x = a[i].first;
  -      ll y = a[i].second;
  -      ll v1 = d[y] + abs(x-y) + l;
  -      ll v2 = u[x] + abs(x-y) + l;
  -      u[x] =  max(u[x],v1); 
  -      ans = max(ans, u[x]);
  -      d[y] = max(d[y],v2);  
  -      ans = max(ans, d[y]);
  -  }


- **어려웠던 점**
  - 그냥 dp는 잘 못 풀겠다. 난 바닥이다.

- **복잡도**
  $(var):
  - 시간복잡도: `O(NlogN)`
  - 공간복잡도: `O(N)`
