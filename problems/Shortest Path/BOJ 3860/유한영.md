**내가 이해한 문제내용**

묘지 탈출.
묘비 / 구멍

 <hr/>
 
**접근 방식**

negative cycle이 있으므로,
각 잔디를 vertex로 하고 잔디간의 거리가 1 인 graph로 
구멍은 새로운 하나의 노드를 추가해주면된다. 
Bellman-ford algorithm


 <hr/>
 
**어려웠던 점**

예제에서 2번째까지는 잘 맞는데 3번째 예제에서 틀리는 데 
시간초과도 아니고, 중간에 아예 틀린 것도 아니고 뭐가 틀렸는지 도저히 모르겠습니다..

<hr/>

**복잡도**
O(WH^2)
