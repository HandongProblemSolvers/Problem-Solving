## BOJ10217 - KCM Travel

- 내가 이해한 문제내용
  - 1부터 N까지 최단거리를 찾는 문제
- 접근 방식
  - DFS를 사용해서 모든 거리를 탐색하되, 비용이 M을 넘거나 길이가 이전에 루트의 최소 길이보다 길어지면 탐색하지 않음.
- 어려웠던 점
  - 코드 구현에 문제가 있는 것인지, 문제 해결 방법을 잘못 생각한건지 잘 모르겠다..
  - 답을 봤는데 다익스트라랑 dp로 풀었던데 나는 dfs로 풀릴 것 같다고 생각하여쓰아 구현실패,,

- 복잡도
  - 시간복잡도:  O(N*N)
