//DP_계단오르기_2579

- 이해한 문제내용
	- 주어진 세가지 규칙에 맞추어 계단을 오를때 얻을 수 있는 최댓값

- 접근방식
	- 마지막 계단을 밝아야 하는 규칙을 성립할 때
		1. 마지막 계단 전 계단을 밝고 올라옴 => dp[n] = dp[n-3] + arr[n-1] + arr[n]
		2. 마지막 계단 전 계단을 밝지 않고 올라옴 => dp[n] = dp[n-2] + arr[n]
	- dp[3]까지는 직접 입력

- 어려웠던점
	- 규칙 하나하나를 보기보다 전체적인 틀에서 마지막 계단을 기준으로 잡는다는 생각

- 시간복잡도 : O(N)
