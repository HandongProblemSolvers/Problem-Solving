# 내가 이해한 문제 내용
메뚜기의 현재위치 (C, R) 이동경로(C1, R1 to C2, R2)는 다음과 같다.
1. |r1-r2| = 1 && |c1-c2| > 1 
2. |c1-c2| = 1 && |r1-r2| > 1

또한 꽃잎의 갯수가 현재 꽃보다 많아야 한다.
위의 3조건을 기반으로 최대한 많이 방문하게 해보자
