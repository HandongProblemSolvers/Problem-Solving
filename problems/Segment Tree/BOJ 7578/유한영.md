**내가 이해한 문제내용**

공장들을 pair할 때, 각 공장들마다 겹쳐지는 선들의 점의 개수를 출력

 <hr/>
 
**접근 방식**

문제를 봤을 때는 O(N^2)이면 바로 풀 수 있어서 절대 아닐 거라 
생각하고 보니 N = 500000이나 되었습니다.
그래서 정렬을 해서 O(NlogN)으로 푸는 방법을 찾았습니다.
일단 입력을 다 받은 후에 sort()를 해주고,
pair로 위와 아래 중에 더 나중에 있는 것으로 check를 해주고
check를 더해줬습니다.
그리고 값을 선에 마지막 값에 따라 빼줬습니다.

 <hr/>
 
**어려웠던 점**

처음에 Map을 사용하여 입력을 받아서 vector에 넣어줬습니다.
Map을 잘 사용하지 않아서 몰랐는데, Map이 정렬하면서 삽입을 해준다는 것을 알고
unordered_map을 사용하였습니다만 몇 개는 정답 잘 나오다가
어느 부분에서 틀린 건 지 맞았습니다는 뜨지를 않네요..
