**내가 이해한 문제내용**

각 기지마다 라디오를 설치한다

기지에는 위성이 연결될 수 있다.

라디오로 연결된 기지들에는 위성이 하나만 연결되면 된다.

 <hr/>
 
**접근 방식**

Greedy + union find

1. 각 기지마다의 거리를 구한다

2. 각 기지마다의 거리를 기준으로 sort

3. sort된 거리들을 greedy로 연결한다. 

4. 연결할 때, union find를 사용한다.

 <hr/>

**복잡도**
O(N*P^2)
