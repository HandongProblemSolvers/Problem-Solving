**접근 방식**

3가지 경우로 나뉜다.

수열 a[N]를 탐색하다가 a[k]>a[k+1]인 부분을 split이라고 정의할 때

1. split이 0인 경우 -> 이미 정렬되어 있으므로 N을 출력해주면 된다.

2. split이 2인 경우 -> 하나를 버려도 정렬되지 않은 부분이 남아 있으므로 0출력

3. split이 1인 경우 

a[k]>a[k+1] 인 부분을 보면

a[k-1] a[k] a[k+1] a[k+2]를 확인 할 때

(* 여기서 a[k-1] < a[k] && a[k]>a[k+1] && a[k+1]<a[k+2]인 것은 확정)

1) a[k-1] <= a[k+1] 이면 a[k]를 버릴 수 있다. 

2) a[k] <= a[k+2] 이면 a[k+1]를 버릴 수 있다.

 1)과 2)각각이 성립할 때마다 경우를 count해준다.
